question:
Write a PL/SQL block of code to calculate the factorial value of number


Output code:

SET SERVEROUTPUT ON;

ACCEPT n NUMBER PROMPT 'Enter a number: '

DECLARE
    num     NUMBER := &n;
    fact    NUMBER := 1;
BEGIN
    FOR i IN 1..num LOOP
        fact := fact * i;
    END LOOP;

    DBMS_OUTPUT.PUT_LINE('Factorial of ' || num || ' is: ' || fact);
END;
/
